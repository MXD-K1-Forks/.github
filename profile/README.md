# MXD-K1-Forks

This organization exists to keep my main GitHub profile — **[MXD-K1](https://github.com/MXD-K1)** — clean and focused on my own original work.

Instead of cluttering my personal account with forked repositories,any repo I fork will exist here. That way, my main profile only shows projects I actually built.

---

## Why a separate org for forks?

GitHub doesn't give you a great way to separate forks from profiles.

- **MXD-K1**: Original projects, experiments, and tools I built myself
- **MXD-K1-Forks**: Forks of other people's repos for any reason.
